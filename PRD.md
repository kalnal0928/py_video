# PRD: Py Video Player

## 개요
간단하고 가벼운 데스크탑 동영상 플레이어(Windows 우선)를 개발한다. Python + PyQt5 UI, `python-vlc`(libVLC)를 사용하여 다양한 코덱을 지원한다.

## 목표
- 사용자가 로컬 동영상 파일을 쉽게 재생할 수 있도록 한다.
- 파일/폴더를 열어 재생목록에 추가하고, 재생목록에서 순서를 선택해서 재생할 수 있다.
- 기본적인 재생 제어(재생/일시정지, 탐색, 볼륨, 전체화면)를 단축키로 제공한다.
- 다양한 코덱을 지원하여 널리 사용되는 포맷을 재생할 수 있도록 한다.

## 핵심 기능
- 파일 열기: 다중 파일 선택을 지원하여 재생목록에 항목 추가
- 폴더 열기: 선택한 폴더(재귀 탐색) 내의 일반적인 비디오 확장자를 가진 파일을 재생목록에 일괄 추가
- 재생목록 창: 우측에 재생목록 표시, 더블클릭 시 재생
- 재생 제어:
  - Enter: 전체화면 토글
  - Space: 재생/일시정지
  - ← / →: -5초 / +5초 탐색
  - ↑ / ↓: 볼륨 +10% / -10%
- 다양한 코덱 지원: `python-vlc`(libVLC)를 백엔드로 사용하여 MKV, MP4, AVI, MOV, FLV, WMV 등 널리 쓰이는 코덱/컨테이너 지원

## 기술 및 의존성
- Python 3.8+
- PyQt5: UI
- python-vlc: VLC의 libVLC를 래핑하여 재생 기능 제공
- (권장) 시스템에 VLC 설치 — libVLC 바이너리 사용 시 재생 안정성 향상

## UI 설계
- 메인 윈도우: 좌측에 비디오 프레임과 하단 컨트롤(열기, 폴더 열기, 재생), 우측에 재생목록
- 재생목록: 파일명 리스트, 더블클릭으로 재생, 향후 이동/삭제/순서 변경 기능 확장 가능

## 파일 형식 필터
초기 단계에서 다음 확장자들을 우선 지원한다: `.mp4, .mkv, .avi, .mov, .wmv, .flv, .webm`.
(libVLC 사용으로 추가 코덱 지원이 가능함)

## 비기능 요구사항
- Windows에서 우선 동작 보장
- 네트워크 스트리밍은 초기 버전에서 제외
- 대용량 재생목록(수백 파일) 처리 가능: UI 스무스함 유지

## 검증 항목 / 수락 기준
- 파일/폴더 열기로 재생목록에 적어도 하나 이상의 지원 포맷 파일이 추가되어야 함
- 재생목록 더블클릭 시 해당 파일이 재생되어야 함
- 단축키(Enter, Space, 방향키)로 동작이 수행되어야 함
- 플레이어가 mp4/mkv 파일을 재생할 수 있어야 함 (libVLC가 설치되어 있어야 함)

## 향후 확장(옵션)
- 재생목록에서 항목 드래그로 순서 변경
- 플레이리스트 저장/로드
- 자막 로드 및 싱크 조정
- 하드웨어 가속 설정 지원

## 배포 및 설치 지침(간단)
- Python 3.8+ 설치
- `pip install -r requirements.txt`
- (권장) VLC 설치
- `python player.py "C:\path\to\video.mp4"`
